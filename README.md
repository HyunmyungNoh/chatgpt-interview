# 면접 예상 질문 생성기
- 사용자가 입력한 정보를 기반으로 하여 면접 예상 질문지를 생성합니다.
- 이에 대해 답변 후 제출하면, 해당 답변에 대해 간단하게 평가도 진행하는 웹 페이지입니다.
- 사용자 입력 값은 면접 예상 질문지에는 지원 분야, 간단한 자기 소개, 회사 정보, 면접 문항 개수이고 면접 평가지애는 (질문), 답변입니다.
